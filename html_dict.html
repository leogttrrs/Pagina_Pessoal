Tags e suas Descrições

Tag Descrição
<!DOCTYPE html> Declara o tipo de documento e a versão do HTML (HTML5, neste caso).
<html> Define o início de um documento HTML e encapsula todo o conteúdo da página.
<head> Contém metadados sobre o documento, como informações de codificação, título e links de estilos.
<meta> Fornece metadados, como o conjunto de caracteres e as configurações de viewport para dispositivos móveis.
<title> Define o título da página, que aparece na aba do navegador.
<link> Relaciona o documento atual a um recurso externo, como uma folha de estilo CSS ou um ícone de favicon.
<body> Contém o conteúdo visível da página, incluindo texto, imagens e links.
<div> Um contêiner genérico que agrupa outros elementos para aplicar estilos ou layout.
<header> Representa um cabeçalho introdutório ou um grupo de links de navegação.
<img> Exibe uma imagem na página. O atributo `alt` fornece um texto alternativo caso a imagem não carregue.
<h1>, <h2> Representam títulos e subtítulos. <h1> é o mais importante, enquanto <h2> é de nível inferior.
<a> Define um hyperlink que permite navegar para outra página ou recurso. O atributo `href` especifica o destino.
<p> Define um parágrafo de texto.
<ul> Cria uma lista não ordenada, com marcadores para os itens da lista.
<li> Define um item em uma lista (tanto em listas ordenadas quanto não ordenadas).
<section> Representa uma seção genérica do documento, que pode ter um título e conteúdo temático relacionado.
<h4> Define um título de nível 4, utilizado para subtítulos dentro de seções.
<footer> Representa o rodapé de uma seção ou página, geralmente contendo informações de copyright ou links.
<main> Define a área principal do conteúdo da página, onde se encontra o conteúdo mais relevante.
<table> Define uma tabela, que é usada para organizar dados em linhas e colunas.
<thead> Define um cabeçalho para a tabela, que geralmente contém títulos para as colunas.
<tbody> Representa o corpo da tabela, que contém as linhas de dados.
<tr> Define uma linha em uma tabela.
<th> Define uma célula de cabeçalho em uma tabela, que geralmente está em negrito e centralizada.
<td> Define uma célula de dados em uma tabela.
